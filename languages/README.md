# Languages

## 概要

プログラミング言語ごとの文法・機能・サンプルコードを整理しています。

---

## 目次

| 言語 | 内容 |
|------|------|
| [Dart](./dart/) | クラス、関数、Null安全、型システム |
| [Java](./java/) | ディープコピー、ファイル操作、リフレクション、STS4 |
| [JavaScript](./javascript/) | TODOサンプルアプリ |
| [PHP](./php/) | （将来用） |
| [VBA](./vba/) | セル操作、CSV生成 |

---

## ディレクトリ構造

```
languages/
├── dart/              # Dart言語リファレンス
│   ├── class/         # クラス関連（継承、Mixin、ジェネリクス等）
│   ├── function/      # 関数（async、例外処理等）
│   ├── null/          # Null安全（late等）
│   └── type/          # 型（List、Map、Set等）
├── java/              # Java言語リファレンス
│   ├── deepCopy/      # ディープコピー
│   ├── file/          # ファイル操作
│   ├── reflection/    # リフレクション
│   └── sts4/          # Spring Tool Suite 4
├── javascript/        # JavaScript
│   └── todo-sample/   # TODOサンプルアプリ
├── php/               # PHP（将来用）
└── vba/               # VBA
```
