# Next.js

## 概要

Next.jsは、Reactベースのフルスタックフレームワーク。サーバーサイドレンダリング（SSR）、静的サイト生成（SSG）、App Routerによる直感的なルーティングなど、モダンなWeb開発に必要な機能を提供する。

## 主な特徴

| 特徴 | 説明 |
|------|------|
| App Router | ファイルベースルーティング、レイアウト、並列ルート |
| Server Components | サーバーサイドでのコンポーネントレンダリング（デフォルト） |
| レンダリング戦略 | SSR / SSG / ISR / CSR を柔軟に選択 |
| Server Actions | サーバーサイドの関数を直接呼び出し |
| 最適化機能 | 画像、フォント、スクリプトの自動最適化 |
| TypeScript | ファーストクラスサポート |
